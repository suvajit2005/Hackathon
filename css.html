üéØ What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It controls how elements are displayed on web pages, including their layout, colors, fonts, and more.

Why Use CSS?

CSS is used to separate content from presentation, which provides greater flexibility and control in the specification of presentation characteristics. It enables multiple web pages to share formatting and reduces complexity and repetition in structural content.

üìù CSS Syntax

CSS consists of style rules that are interpreted by the browser and applied to the corresponding elements.

Basic CSS Syntax Structure

selector {
    property: value;
    property: value;
}

Example:

p {
    color: blue;
    font-size: 16px;
}

In this example, all paragraph elements will have blue text with a font size of 16 pixels.

üîç CSS Selectors

CSS selectors are patterns used to select the element(s) you want to style.

Basic Selectors

  • Element selector: Selects elements based on the element name (e.g., p, div, h1)
  • Class selector: Selects elements with a specific class attribute (e.g., .my-class)
  • ID selector: Selects a single element with a specific id attribute (e.g., #my-id)
  • Universal selector: Selects all elements (*)

üé® Working with Colors in CSS

CSS provides several ways to specify colors, each with its own use cases and advantages.

Color Representation Methods

  • Named colors: Predefined color names (e.g., red, blue, green)
  • Hexadecimal colors: Six-digit codes representing RGB values (e.g., #FF0000 for red)
  • RGB: Functional notation using red, green, blue values (e.g., rgb(255, 0, 0))
  • RGBA: RGB with alpha channel for transparency (e.g., rgba(255, 0, 0, 0.5))
  • HSL: Hue, Saturation, Lightness model (e.g., hsl(0, 100%, 50%))
  • HSLA: HSL with alpha channel for transparency

üîÑ Types of CSS

There are three ways to apply CSS to HTML documents, each with different use cases and levels of specificity.

Inline CSS

Applied directly to HTML elements using the style attribute. Has the highest specificity.

<p style="color: blue; font-size: 16px;">This is a paragraph with inline styles.</p>

Internal CSS

Defined within the <style> tags in the <head> section of an HTML document. Applies to the entire document.

<head>
    <style>
        p {
            color: blue;
            font-size: 16px;
        }
    </style>
</head>

External CSS

Defined in a separate .css file and linked to the HTML document using the <link> tag. Most efficient method for styling multiple pages.

<head>
    <link rel="stylesheet" href="styles.css">
</head>

üì¶ CSS Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree. Each box has content, padding, border, and margin areas.

Box Model Components

  • Content: The actual content of the box, where text and images appear
  • Padding: Clears an area around the content; transparent
  • Border: A border that goes around the padding and content
  • Margin: Clears an area outside the border; transparent

Box-Sizing Property

The box-sizing property controls how the total width and height of an element is calculated.

  • content-box: Default. Width and height only apply to the content
  • border-box: Width and height apply to content, padding, and border

üìç CSS Positioning

The position property specifies the type of positioning method used for an element.

Position Values

  • static: Default. Elements are positioned according to the normal flow of the document
  • relative: Positioned relative to its normal position
  • absolute: Positioned relative to the nearest positioned ancestor
  • fixed: Positioned relative to the browser window
  • sticky: Toggles between relative and fixed based on scroll position

üìè CSS Flexbox

Flexbox is a CSS layout module that makes it easier to design flexible and responsive layout structures.

Why Use Flexbox?

  • Easier alignment of elements
  • Distribute space dynamically between elements
  • Handle different screen sizes and display devices
  • Simplify complex layouts

Flex Container Properties

These properties are applied to the parent element (flex container):

  • display: Defines a flex container (flex or inline-flex)
  • flex-direction: Defines the direction of flex items (row, row-reverse, column, column-reverse)
  • flex-wrap: Specifies whether flex items should wrap (nowrap, wrap, wrap-reverse)
  • justify-content: Aligns flex items along the main axis (flex-start, flex-end, center, space-between, space-around, space-evenly)
  • align-items: Aligns flex items along the cross axis (stretch, flex-start, flex-end, center, baseline)
  • align-content: Aligns flex lines when there is extra space in the cross axis (flex-start, flex-end, center, space-between, space-around, stretch)
