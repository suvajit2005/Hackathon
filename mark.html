üîç What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure of a web page and consists of a series of elements that tell the browser how to display content.

Key Points about HTML:

  • HTML stands for HyperText Markup Language
  • HTML is the standard markup language for Web pages
  • HTML elements are the building blocks of HTML pages
  • HTML elements are represented by tags
  • HTML tags label pieces of content such as "heading", "paragraph", "table", etc.
  • Browsers do not display HTML tags but use them to render content

Simple HTML Example:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>
    <h1>My First Heading</h1>
    <p>My first paragraph.</p>
</body>
</html>

üìÑ HTML Boilerplate

An HTML boilerplate is a template you will include at the beginning of your HTML projects. It contains the essential parts of an HTML document that you need to set up before adding your content.

Basic HTML5 Boilerplate:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <!-- Your content goes here -->
</body>
</html>

Explanation of Boilerplate Elements:

Element Purpose
<!DOCTYPE html> Defines the document type and HTML version (HTML5)
<html lang="en"> Root element with language specification
<head> Contains meta information about the document
<meta charset="UTF-8"> Defines character encoding for the document
<meta name="viewport"> Ensures proper rendering and zooming on mobile devices
<title> Specifies the title of the document (shown in browser tab)
<body> Contains the visible page content

üîó Anchor Tag

The HTML <a> tag (anchor tag) defines a hyperlink that is used to link from one page to another. The most important attribute of the <a> element is the href attribute, which indicates the link's destination.

Basic Syntax:

<a href="url">link text</a>

Examples of Anchor Tags:

  • Link to an external website: Visit Example.com
  • Link to another page on the same site: About Us
  • Link to an email address: Send Email
  • Link to a phone number: Call Us
  • Link to a section on the same page: Go to What is HTML

Anchor Tag Attributes:

Attribute Description
href Specifies the URL of the page the link goes to
target Specifies where to open the linked document (_blank, _self, _parent, _top)
download Specifies that the target will be downloaded when clicked
rel Specifies the relationship between current and linked document

üìÅ File Paths in HTML

File paths describe the location of a file in a website's folder structure.

Types of File Paths:

Path Type Description Example
Absolute Path Full URL to an external resource https://www.example.com/images/pic.jpg
Relative Path Path relative to the current page images/pic.jpg or ../images/pic.jpg
Root Relative Path Path relative to the root of the website /images/pic.jpg

Relative Path Examples:

  • src="image.jpg" - The image is located in the same folder as the current page
  • src="images/image.jpg" - The image is located in the images folder in the current folder
  • src="../image.jpg" - The image is located in the folder one level up from the current folder
  • src="/images/image.jpg" - The image is located in the images folder at the root of the current web

Path Examples in Code:

<!-- Absolute path -->
<img src="https://www.example.com/images/photo.jpg" alt="Example">

<!-- Relative path -->
<img src="images/photo.jpg" alt="Example">

<!-- Relative path going up one level -->
<img src="../images/photo.jpg" alt="Example">

<!-- Root relative path -->
<img src="/images/photo.jpg" alt="Example">

üì¶ Div, Class & ID in HTML

These are important attributes used for styling and JavaScript manipulation of HTML elements.

The Div Element:

The <div> tag defines a division or a section in an HTML document. It is used as a container for other HTML elements.

This is a div element

It can contain any other HTML elements

Class Attribute:

The class attribute is used to specify one or more class names for an element. Classes can be used by CSS and JavaScript.

This paragraph has a class="highlight"

This paragraph has two classes: class="highlight important"

ID Attribute:

The id attribute specifies a unique id for an HTML element. The value must be unique within the HTML document.

This paragraph has a unique id="special-paragraph"

Code Examples:

<!-- Div element -->
<div>
    <h2>Section Title</h2>
    <p>Some content here</p>
</div>

<!-- Class attribute -->
<p class="highlight">This is highlighted text</p>
<p class="highlight important">This is important highlighted text</p>

<!-- ID attribute -->
<div id="header">This is the header section</div>
<p id="intro">Introduction paragraph</p>

Differences Between Class and ID:

Class ID
Can be used on multiple elements Must be unique to a single element
Selected with .classname in CSS Selected with #idname in CSS
An element can have multiple classes An element can have only one ID
Used for styling groups of elements Used for unique styling and JavaScript manipulation

üéµ Media Tags in HTML

HTML provides several tags for embedding multimedia content like audio, video, and external content from sites like YouTube.

Audio Element:

The <audio> tag is used to embed sound content in a document.

Video Element:

The <video> tag is used to embed video content in a document.

Embedding YouTube Videos:

You can embed YouTube videos using the <iframe> tag.

Media Code Examples:

<!-- Audio -->
<audio controls>
    <source src="audiofile.mp3" type="audio/mpeg">
    Your browser does not support the audio element.
</audio>

<!-- Video -->
<video width="320" height="240" controls>
    <source src="movie.mp4" type="video/mp4">
    <source src="movie.ogg" type="video/ogg">
    Your browser does not support the video tag.
</video>

<!-- YouTube Video -->
<iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" title="YouTube video player" frameborder="0" allow="accelerometer; autoplay;
 encrypted-media; 
allowfullscreen></iframe>

Media Attributes:

Attribute Description
controls Displays audio/video controls (play, pause, volume, etc.)
autoplay Specifies that the audio/video will start playing as soon as it is ready
loop Specifies that the audio/video will start over again, every time it is finished
muted Specifies that the audio output of the video should be muted
preload Specifies if and how the author thinks the video should be loaded when the page loads
poster Specifies an image to be shown while the video is downloading, or until the user hits the play button
